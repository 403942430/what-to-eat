# 方案变更：美团外卖 → 美团外卖

## 变更原因
美团外卖无 H5 网页版，无法通过用户脚本抓取评论和订单。
美团外卖有完整 H5 网页版（h5.waimai.meituan.com），支持 Tampermonkey 脚本。

## 变更内容
1. 用户脚本目标从美团外卖改为美团外卖 H5
2. 平台标识从"美团外卖"改为"美团外卖"
3. 店铺分类适配美团外卖（新增：汉堡、蛋糕、甜品等）
4. 测试数据改用美团外卖常见店铺名

## 不变内容
- PWA 前端架构：完全不变
- 推荐引擎 + 贝叶斯修正：不变
- 后端 NLP + 本地规则引擎：不变
- 部署方案（Vercel + Railway）：不变
